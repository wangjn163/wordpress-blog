### 🐛 遇到的问题及解决

**问题1：分页页面"未找到匹配的模板"**
- 原因：主题缺少index.php模板
- 解决：创建了完整的模板文件

**问题2：内存不足（Allowed memory exhausted）**
- 原因：PHP默认128MB不够
- 解决：增加到512MB

**问题3：AI评论自动回复不工作**
- 原因：函数重复定义导致致命错误
- 原因：访客评论user_id=0时的检查逻辑问题
- 解决：优化代码，添加双重机制（即时+定时）

**问题4：微信"继续访问"提示**
- 原因：WordPress的重定向和安全检查
- 解决：禁用canonical重定向，添加微信友好的HTTP头

**问题5：评论3天未收到AI回复**
- 原因：评论发表时AI系统还未部署
- 解决：手动补充历史评论回复

**问题6：WordPress定时任务依赖网站访问**
- 原因：WP-Cron是伪定时任务，需要用户访问网站才触发
- 解决：设置系统级cron任务，每5分钟访问wp-cron.php确保定时任务执行

**问题7：未回复评论检查和修复**
- 需求：需要监控AI回复率并快速发现问题
- 解决：为wordpress-test技能添加了4个新命令（11-14），支持回复率验证、未回复评论列表、详情查看和定时任务检查
